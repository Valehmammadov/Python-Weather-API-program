# WeatherAPI Python Example 🌦️

Simple Python application demonstrating how to securely access the **WeatherAPI** service using an API key stored in a `.env` file.

---

## 📌 Features

- Secure API key management via `.env`
- Uses `requests` for HTTP calls
- Retrieves current weather data
- Beginner-friendly structure

---

## 🚀 Requirements

- Python 3.x
- `requests`
- `python-dotenv`

Install dependencies:

```bash
pip install requests python-dotenv
